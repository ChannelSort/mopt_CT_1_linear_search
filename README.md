# mopt_CT_1_linear_search

Лабораторная работа по методам одномерного поиска.

Сделал Скелет для кода (пока не рефакторил, но реализации 1 в 1 относительно конспекта) и шаблон отчета в LaTeX.

## Содержимое

- `lab1_linear_search.py` - OOP-каркас лабораторной работы.
- `Report.tex` - базовый шаблон отчета.
- `outputs/plots` - папка для сохраняемых графиков.
- `outputs/tables` - папка для сохраняемых таблиц.

## Что уже готово

- базовые абстрактные классы для функций и оптимизаторов;
- реализованы `Passive search` и `Brent (SciPy)`;
- добавлены две унимодальные тестовые функции;
- добавлен класс `OptimizationExperiment`;
- добавлены функции построения графиков и выгрузки таблиц;
- оставлены `TODO`-заглушки для остальных методов и моделей.
